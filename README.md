# Lollo176ITA Website

This repository contains the source code of my personal website built with **React** and **Tailwind CSS**.

## Requirements
- Node.js 18 or later
- npm

## Installation

```bash
npm install
```

## Development

Start the development server with hot reload:

```bash
npm start
```

## Testing

Run the test suite with:

```bash
npm test
```

## Production Build

Create an optimized build in the `build` directory:

```bash
npm run build
```

## Deployment

The project is configured to deploy to GitHub Pages. Use the following command to build and publish the site:

```bash
npm run deploy
```
